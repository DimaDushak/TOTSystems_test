# TOTSystems_test

Приложение - корпоративный чат от лица одного из сотрудников, состоящий из 2 разделов (чат для обсуждения рабочих вопросов и чат для общения на любые темы).

При запуске приложение запрашивет логин и пароль. После успешного ввода данных пользователь может:
  - писать сообщения, в том числе используя смайлики;
  - отправлять сообщения;
  - редактировать сообщения;
  - удалять сообщения.
  
Начальный список сообщений берётся из статического файла, отправленные сообщения хранятся в памяти браузера (используется localStorage).
  
Приложение написано с использованием библиотеки React. Используются только функциональные компоненты и хуки.
Для рефакторинга и проверки данных используeтся Typescript.

Для запуска приложение используйте команду npm run dev.
